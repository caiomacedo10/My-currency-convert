# Conversor de Moedas Completo 

## Descrição  
Projeto de um conversor de moedas que permite a conversão dinâmica entre diversas moedas, incluindo Real (BRL), Dólar (USD), Euro (EUR), Libra (GBP) e Bitcoin (BTC). Desenvolvido com foco em precisão, usabilidade e atualização em tempo real das taxas de câmbio, o projeto oferece uma interface simples e intuitiva para facilitar a experiência do usuário.

## Tecnologias Utilizadas  

- HTML5 para estruturação da página  

- CSS3 para estilização responsiva e clean  

- JavaScript para lógica da conversão e manipulação dos dados  

- (Se aplicável) API externa para atualização das taxas de câmbio em tempo real

## Funcionalidades  

- Conversão precisa e instantânea entre as moedas suportadas  

- Interface limpa e fácil de usar, com campos para inserção de valores e seleção de moedas  

- Atualização automática das taxas de câmbio para refletir valores atuais (incluir detalhes da API se houver)  

- Suporte para moedas tradicionais e criptomoeda (Bitcoin)
